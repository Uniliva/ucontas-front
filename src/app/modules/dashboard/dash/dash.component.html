Contextualizando

Este é um projeto que tem como finalidade demostrar as habilidade com:

• Angular 8+
• Java 8+
• Rest
• Spring Boot
• JUnit
• Oracle 12g
• Docker

Ele está dividido e 3 partes.

Parte 1: Ucontas-api

Foi criada uma api rest simples para contas com categorização, utilizando:

  • Spring boot
  • Java 8
  • Spring Jpa
  • Oracle 12c
  • h3 para demostração testes
  • Junit
  • Api de consulta cep, para demostração de serviços rest
  • Swagger
  • CodeCov para analise de cobertura de teste
  • Travis CI pra build e geração de image docker

Links uteis para o projeto:

  • Projeto github : https://github.com/Uniliva/ucontas-api
  • Swagger : https://ucontas-api.herokuapp.com/swagger-ui.html
  • Projeto no Travis : https://travis-ci.com/Uniliva/Ucontas
  • CodeCov : https://codecov.io/gh/Uniliva/Ucontas

Parte 2: Front-end

Este é uma palicação simples utilizabdo angular8+, que consome a api descrita anteriomente

Parte 3: Dockerizado

Para demostrar a parte de docker, foi criado images com os projetos descritos anteriormente, e para facilitar a implatação foi criado um projeto a parte que faz a "instalação" do banco, da api e do front, usando docker-compose.

Durante build realizado pelo Travis CI, e são criadas imagens que são enviadas para o docker hub.

Para mais detalhes de uma olhada no projeto.

  • Projeto github : https://github.com/Uniliva/ucontas-installer
  • Imagem da api : https://hub.docker.com/repository/docker/uniliva/ucontas-api
  • Imagem do banco : https://hub.docker.com/repository/docker/uniliva/oracle_db
